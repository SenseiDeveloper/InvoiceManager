# InvoiceManager
Use:
- Angular 7.2;
- Node server (express);
- Material;
- Bootstrap;
- Drag and drop;

# Start
- git clone https://github.com/SenseiDeveloper/InvoiceManager;
- npm i;
- npm start (start angular project);
- cd server;
- node server (start node express server);
